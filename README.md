## 💡 PR Convention

| 아이콘 | 코드                          | 설명                                      |
| ------ | ----------------------------- | ----------------------------------------- |
| ✨     | `:sparkles:`                  | 새로운 기능 추가                          |
| 💄     | `:lipstick:`                  | UI/스타일 관련 파일 수정                  |
| 🐛     | `:bug:`                       | 버그 수정                                 |
| 📝     | `:memo:`                      | 문서 작성 또는 수정 (예: README, Wiki 등) |
| 🎨     | `:art:`                       | 코드의 구조/형태 개선                     |
| ⚡️    | `:zap:`                       | 성능 개선                                 |
| 🔥     | `:fire:`                      | 불필요한 코드/파일 삭제                   |
| 🚑     | `:ambulance:`                 | 긴급 수정 (핫픽스 등)                     |
| ⏪     | `:rewind:`                    | 변경 내용 되돌리기                        |
| 🔀     | `:twisted_rightwards_arrows:` | 브랜치 병합                               |
| 💡     | `:bulb:`                      | 주석 추가/수정                            |
| 🗃      | `:card_file_box:`             | 데이터베이스 관련 작업                    |
| 🔧     | `:wrench:`                    | 설정 파일 수정                            |
| ✅     | `:white_check_mark:`          | 테스트 코드 추가/수정                     |
| 📦     | `:package:`                   | 패키지/라이브러리 관련 변경               |

### 💬 PR 제목 예시:

```text
✨ 로그인 기능 구현
🐛 토큰 갱신 오류 수정
```
